Description

This asset provides a comprehensive audio management system for Unity games, handling background music, sound effects, and UI sounds with advanced features.

It offers robust audio control with minimal coding required while maintaining flexibility for complex audio scenarios.

Includes:

  • Audio Manager Prefab
  • AudioManager Class Script
  • Sound Class Script
  • AudioScheduler Integration
  • Documentation and examples

Features

  • Single-line audio playback
  • Multiple mixer groups (SFX, UI, Music)
  • Cross-scene persistence
  • Volume persistence between sessions
  • One-shot playback support
  • Dynamic audio routing
  • Comprehensive debugging tools

Quick Start

// Play background music
AudioManager.Instance.Play("BackgroundMusic");

// Play UI sound effect
AudioManager.Instance.Play("ButtonClick");

// Play Player Hurt sound
AudioManager.Instance.PlayOneShot("PlayerHurt");

// Fade out music smoothly
StartCoroutine(AudioManager.Instance.FadeOut("BackgroundMusic", 2f));

Easy drag and drop of Audio Clips with manual naming required.

UI Player Enemy Weapon Item

API Reference

Core Methods

Play(string name)
Play sound with cooldown checks
PlayOneShot(string name)
Play sound without cooldown checks
Stop(string name)
Stop playing sound

Installation & Configuration

  1. Import the Audio Manager package into your Unity project
  2. Drag the AudioManager prefab into your initial scene
  3. Configure your sounds in the Inspector window
  4. Set up Audio Mixer groups for SFX, UI, and Music
  5. Start playing sounds with simple API calls

Best Practice: Use descriptive names for sounds (e.g., "Player_Jump" not "Sound1")
